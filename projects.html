StickyNote Scrapbook

  • HTML
  • CSS
  • PHP
  • Materialize
  • MySQL
  • Heroku

This website allows the user to store their favourite things from each category in the form of sticky-note-like posts, as well as keep a journal. PHP and MySQL are used to store and recall the posts. It is styled using a combination of pure CSS and Materialize. From this project, I increased my confidence with PHP and learned to balance Materialize with my own CSS to create my particular design. It was a project I had planned and designed all aspects of before begining construction, so I learned to organise the build into components that could be completed in each session.
